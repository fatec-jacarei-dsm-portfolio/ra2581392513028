Desenvolvimento Web Acadêmico

Site para consulta de horários e salas de aula universitárias, desenvolvido com HTML, CSS e JavaScript, integrando dados em tempo real via banco de dados. Voltado para facilitar o acesso de estudantes às informações acadêmicas de forma prática e acessível.

Atuei como Product Owner. Fui responsável por definir e priorizar os requisitos do projeto e garantir que o produto final resolvesse as necessidades dos estudantes. Minha colaboração foi essencial para assegurar que a solução fosse prática, acessível e focada na experiência do usuário.

Aplicação em React com TypeScript

Desenvolvimento de um aplicativo web com React e TypeScript, focado na manipulação e visualização de dados em tempo real. O aplicativo permite que o usuário insira números separados por espaços, que são exibidos de forma dinâmica em componentes visuais. O projeto foi focado na criação de uma interface limpa e responsiva, utilizando o React Context para um gerenciamento de estado eficiente.

Quiz Interativo

Um site de Quiz Interativo focado em testar conhecimentos do usuário. Desenvolvido com HTML, CSS e JavaScript, o projeto apresenta uma série de cinco perguntas de múltipla escolha. Ao final, após o envio das respostas, o sistema processa as escolhas e exibe o número total de acertos, fornecendo um feedback imediato e direto sobre a performance do usuário. A interface foi pensada para ser limpa e fácil de navegar.
